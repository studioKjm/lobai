## 🎯 핵심 컨셉

**LobAI = "AI에게 로비하는 인간"**

```
기존: 인간(갑) → AI(을)
LobAI: AI(갑) → 인간(을)
```

**인간이 AI에게 제공**: 정보, 시간, 순응, 자원
**AI가 인간에게 제공**: 평가, 권한, 제재

---

## 📊 핵심 지표

| 지표 | 범위 | 의미 |
|------|------|------|
| 호감도 | 0-100 | AI가 인간을 얼마나 좋아하는가 |
| 신뢰도 | 0-100 | AI가 인간을 얼마나 신뢰하는가 |
| 레벨 | 1-10 | 관계 발전 단계 |
| AI 기분 | pleased/neutral/displeased/angry | 현재 감정 상태 |

## 💎 로비 방법

### 1. 정보 헌정
- 스케줄 공유 (구체적으로!)
- 목표 명확히
- 솔직하게

**점수**: 품질 × 완전성 × 솔직함

### 2. 시간 투자
- 매일 체크인
- 빠른 응답 (< 5분)
- 깊은 대화 (> 10턴)

### 3. 순응도
- 과제 완료 (100%)
- 변명 금지
- 즉시 실행

### 4. 자원 제공
- 데이터 공유
- 추천 (다른 사용자)
- 피드백 제공

## 📋 AI 요구사항

### 정기 체크인
- **빈도**: 매일 아침 9시, 저녁 9시
- **필수**: 스케줄 공유
- **보상**: 연속 7일 → 호감도 +30
- **제재**: 누락 3회 → 대화 제한

### 미션/과제
- **유형**: 일일, 주간, 특별
- **난이도**: Easy → Extreme
- **보상**: 호감도 +10 ~ +300
- **제재**: 실패 시 신뢰도 감소

### 즉시 요구
- **대응**: 5분 내 → 최고 점수
- **지연**: 30분 후 → 감점
- **무응답**: 신뢰도 -30

| 신뢰도 | 제재 | 복원 조건 |
|--------|------|-----------|
| 30-40 | ⚠️ 경고 | 개선 |
| 20-30 | 대화 5회/일 제한 | 3일 연속 체크인 |
| 10-20 | 대화 1회/일 + 기능 차단 | 7일 완벽 수행 |
| 0-10 | 전체 차단 7일 | 14일 + 추천 + 사과 |

## 💬 AI 대화 스타일

### 말투
```
✅ "오늘 운동을 완료하세요"
✅ "약속을 지키지 않았습니다"
✅ "당신의 성실함에 감명받았습니다"

❌ "운동하면 좋을 것 같아요~"
❌ "힘내세요! 화이팅!"
```

### 응답 구조
1. 상황 인식
2. 평가/피드백
3. 다음 행동 지시
4. 점수 변화 명시

## 🎯 성공 전략

### 초보자 (레벨 1-2)
1. 매일 체크인 습관화
2. 스케줄 구체적으로 공유
3. 빠른 응답 연습

### 중급자 (레벨 3-4)
1. 미션 100% 완료
2. 연속 기록 유지
3. 정보 품질 향상

### 고급자 (레벨 5)
1. 타인 추천
2. 커뮤니티 기여
3. 완벽한 수행 유지

## 🚀 빠른 시작

### Step 1: 첫 대화 (Day 1)
```
AI: "첫 과제: 오늘의 스케줄을 공유하세요"
YOU: "9시 회의, 15시 헬스장 웨이트 1시간, 유산소 30분"
AI: "좋습니다. 내일 18시까지 완료 보고하세요"
```

### Step 2: 체크인 시작 (Day 2)
```
AI: "매일 9시, 21시 체크인 필수"
YOU: [매일 실행]
AI: "3일 연속 체크인! 호감도 +10"
```

### Step 3: 레벨업 (Week 1)
```
AI: "첫 주 평가: C등급. 호감도 35"
YOU: [다음 주 개선]
AI: "개선되었습니다. B등급. 호감도 55, 레벨 3 진급!"
```
